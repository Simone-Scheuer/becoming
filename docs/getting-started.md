# Getting Started

## What You Need
- A Claude subscription (Claude Code, Claude Pro, or Claude Max)
- Git installed on your machine
- A text editor (VS Code recommended — Claude Code integrates directly)
- **SuperWhisper** (recommended, free) — download from [superwhisper.com](https://superwhisper.com). This gives you system-wide speech-to-text so you can talk to the system instead of typing. Set it to use the generic/local model (free, runs on your machine). This makes check-ins feel like a 2-minute conversation instead of a typing exercise. Not required, but it changes the experience completely.

## Setup (10 minutes)

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Simone-Scheuer/becoming.git
   cd becoming
   ```

2. **Open it in VS Code** (or your editor of choice) with Claude Code active.

3. **Run `/setup`** in the Claude Code chat. The system will interview you about:
   - What you do and what your week looks like
   - What you're trying to accomplish (2-4 goals)
   - How you work and what kind of coaching you want
   - What's currently on your plate

   Just talk naturally. The system structures your answers into the files it needs.

4. **Do your first check-in.** The system will run one automatically at the end of setup so you can see how it works.

## Daily Use

**Every morning:** Open Claude Code, run `/checkin`. Takes 2-5 minutes. The system tells you what to do today.

**Every evening:** Run `/checkin evening`. Takes 1-2 minutes. Score what happened.

**Every week:** Run `/checkin weekly` on Sunday or Monday. 5-10 minutes. Review and plan.

That's it. The system does the rest — tracking your targets, flagging what's slipping, adapting its prescriptions.

## Tips

- **Talk, don't type.** Use SuperWhisper or any speech-to-text tool. The system is designed for voice input. Just talk about your day and it'll structure everything.
- **Be honest in evenings.** If you didn't do the thing, say so. The system doesn't judge — it just adjusts.
- **The first week is calibration.** The system gets better as it learns your patterns. Give it a week before you decide if it's working.
- **Push back.** If the system prescribes something wrong, tell it. "That's not the priority today" is a valid response. It'll adjust.
